dependencies
------------

Generally, most notebooks use...

* `qgrid` (requires manual step)
* `ipywidgets` (requires manual step)
* `pandas`
* `plotnine`

However, all dependencies can be found in `requirements.txt`, and installed using...

```
pip install -r requirements.txt
```
